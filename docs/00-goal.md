# Oneclick

## Goal
Build a self-service internal developer platform that allows engineers to:
- Provision services using Git
- Deploy automatically using GitOps
- Monitor their services out of the box
- Operate them safely and securely

This project is a learning and portfolio platform.

## Scenario
- Developer: wants fast deployment, no Kubernetes knowledge.
- Platform Engineer: wants safety, automation, and reliability.

## Success Criteria
- New service onboarded in <10 minutes
- Zero manual kubectl changes
- Full monitoring automatically enabled
- Secure by default
- Reproducible from scratch
